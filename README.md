# ReaSRTBrowser

`ReaSRTBrowser` は、SRT形式の字幕ファイルを REAPER 上で扱いやすくするための ReaScript です。  
字幕アイテムの閲覧、検索、タグ編集、お気に入り管理、音声プレビュー、タイムライン挿入、ライブラリ管理を 1 つの GUI で行えます。

## 主な用途

- 字幕アイテムを一覧で確認する
- 字幕テキストを検索する
- タグやお気に入りを付けて管理する
- 対応する音声をプレビューする
- 選択したアイテムを REAPER のタイムラインへ順番に挿入する
- 複数の SRT をライブラリとしてまとめて横断的に扱う

## 動作要件

- REAPER
- ReaImGui
- SWS Extension

補足:

- スクリプトの実行には `ReaImGui` 及び `SWS Extension` が必要です
- ReaPack 経由でインストールが可能です

## インストール

配布時の想定ファイル構成:

```text
ReaSRTBrowser.lua
reasrt/
resources/
```

インストール手順:

1. `ReaSRTBrowser-vX.X.X.zip` をダウンロードして適当なフォルダに展開し、 `ReaSRTBrowser.lua`、`reasrt` フォルダ、`resources` フォルダを同じ階層のまま配置します。
2. REAPER の `Actions` から `Load ReaScript...` を選び、`ReaSRTBrowser.lua` を読み込みます。
3. 必要に応じて Action List からショートカットやツールバー登録を行います。

初回起動時に、設定ファイルやメタデータ保存用フォルダが自動作成されます。

## 基本的な使い方

- スクリプトを実行すると、`ReaSRTBrowser` のGUIが起動します。

### 1. SRT を読み込む

- 左ペインの `字幕ファイル一覧` で字幕ファイルを管理します
- `ファイル` メニューから字幕ファイルを追加します
- 複数の字幕ファイルをまとめて追加することもできます

### 2. 音声を関連付ける

- 上部の `音声を開く...` から音声ファイルを関連付けできます
- 字幕ファイル読み込み時、同じフォルダに同名または類似名の `.wav` がある場合は自動で関連付けを試みます

### 3. 字幕アイテムを確認・検索する

- 中央の `アイテム一覧` に字幕アイテムが表示されます
- テキスト検索、タグ検索、`お気に入りのみ` フィルタが使えます
- `話者ラベルを隠す` を有効にすると、先頭の話者表記がある場合にそれらを隠した状態で一覧表示・検索できます

### 4. 字幕アイテムを編集する

- 下部の `編集` パネルで字幕アイテムに関する次の内容を編集できます
- `お気に入り`
- `タグ`
- `Global Offset`

### 5. タイムラインへ挿入する

- アイテムをダブルクリック、または選択して `Enter` を押すと、現在のカーソル位置に挿入します
- 複数選択した場合は、選択順に連続して挿入されます

### 6. 音声プレビューを再生する

- アイテムを選択して `Space` を押すと、対応する音声範囲をプレビューします
- `設定` メニューの `プレビュー音量...` からプレビュー音量を 0 から 100 の範囲で変更できます


## その他の機能

### ライブラリ機能

- 左ペインの `ライブラリ` タブではライブラリを作成して管理できます
- 複数の字幕ファイルをまとめて 1 つのライブラリとして扱えます
- ライブラリを開くと、登録した複数の字幕ファイルの内容を横断して検索・参照できます

### 話者タグ追加

- `編集` メニューから`話者タグを追加` を実行できます。
- 字幕の最初に（話者）のような記述が含まれている場合に、自動でタグに登録できます。


## 保存される情報

スクリプトは設定やメタデータを、現在使用中の REAPER リソースフォルダ配下の `ReaSRTBrowser` フォルダへ保存します。  
保存内容には次のようなものが含まれます。

- 最後に開いた字幕ファイル / ライブラリ
- UI レイアウト
- 言語設定
- フォント設定
- プレビュー音量
- 各字幕ファイルに対するタグ / お気に入り / オフセット / 音声関連付け

## 注意事項

- 読み込んだ字幕ファイルの名前を変更したり場所を移動したりすると、登録したタグやお気に入りなどの情報は失われます
- 音声関連付けや自動検出は主に `.wav` を前提としています。`.mp3` 形式などの場合は事前に変換が必要です
- タイムラインへの挿入には、関連付けられた音声ファイルが必要です
